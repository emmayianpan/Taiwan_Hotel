Project Details / Resources
The goal for this project is to analyze Taiwan hotel market and to visualize data in a monthly basis for hoteliers to review competitors performance, pricing and seasonality from macro and micro prospectives, and for potential investors to receive basic information of current market trend in hotel industry.
  • Project Details/Resources

Data Resource

  • Taiwan Hotels Database

    The website, which is operated under Tourism Bureau, Ministry of Transportation and Communications provides month-end overall performace of Taiwan hotels. The reports can be download as PDF or ODF files.

    • ETL Processing

  • Pull raw data from Taiwan Hotels Database and downloaded reports as ODF files. Organized raw data for the first step
  • Transfered into csv files, cleaned/sorted data with correct formatting and created new columns by calculation including RevPAR, Demand, etc. to complete data processing.
  • Loaded data onto PostgreSQL and merged monthly data.
  • Created another dataset for area performance by using combined data to calculate on Jupyter Notebook.
  • Manually input data "Hotle English Name", "Address", "Longtitude", "Latitude" and "Website" from Google for Hotel Map.
  • Exported all data into csv files while the ETL processing was completed.

    • Data Visulizations and Flask Deployment

  • Forked baotm/admindek GitHub Repository and modified its HTML/CSS to create the website structure.
  • Visulize the overview/area/hotel performance data by using JavaScript Plotly.
  • Visualize the hotel location/information by using JavaScript Leaflet and Market Cluster.
  • Present and Publish on the website by using Python Flask and Heroku
